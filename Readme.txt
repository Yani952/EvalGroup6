#  Projet de Gestion par Branches Git

##  Description
Ce projet a pour objectif d’organiser le développement d’un projet en utilisant le système de **branches Git**.  
Chaque branche représente une **partie spécifique du projet**, développée indépendamment avant d’être intégrée à la branche principale.

Cette méthode permet :
- Un meilleur travail en équipe  
- Une organisation claire du code  
- Une réduction des conflits lors du développement  

---

##  Objectifs du Projet
- Créer une **branche par fonctionnalité ou par partie**
- Travailler chaque partie de manière indépendante
- Fusionner les branches dans la branche principale (`main` ou `master`)
- Apprendre et maîtriser les bases du travail collaboratif avec Git

---

##  Organisation des Branches
- `main` / `master` : branche principale, version stable du projet

- `branche-2` :
